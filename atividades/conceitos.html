Conceitos

As entregas desta disciplina recebem conceitos em vez de notas numéricas. Embora cada entrega tenha uma rubrica diferente, todas seguem o sistema descrito abaixo.

Cabe observar que essa descrição é específica para esta disciplina e não representa um sistema obrigatório para todas as disciplinas do Insper ou mesmo da Engenharia. Os professores estabelecem regras de acordo com as necessidades de cada contexto.

Conceito I

O conceito I é atribuído aos casos em que a entrega não foi feita, desviou do objetivo ou não exigiu esforço significativo. Esse conceito é insuficiente para aprovação.

Conceito D

O conceito D é atribuído aos casos em que a entrega foi feita, não desviou do objetivo e exigiu esforço significativo, mas o essencial do objetivo não foi atingido. Esse conceito é insuficiente para aprovação.

Um arredondamento para C pode ser considerado se o essencial não foi atingido por muito pouco e características significativas de B podem ser identificadas.

Conceito C

O conceito C é atribuído aos casos em que o essencial do objetivo foi atingido, mas o desempenho foi abaixo do esperado. Esse conceito é suficiente para aprovação, mas o aluno corre risco significativo de obter I ou D em entregas mais difíceis.

Conceito C+

O conceito C+ é atribuído aos casos em que C foi atingido sem arredondamento e B não foi atingido, mas características significativas de B podem ser identificadas.

Um arredondamento para B pode ser considerado se o desempenho foi muito próximo do esperado e características significativas de A podem ser identificadas.

Conceito B

O conceito B é atribuído aos casos em que o desempenho foi dentro do esperado. Esse conceito é a referência que o aluno deve ter ao realizar a entrega.

Conceito B+

O conceito B+ é atribuído aos casos em que B foi atingido sem arredondamento e A não foi atingido, mas características significativas de A podem ser identificadas.

Conceito A

O conceito A é atribuído aos casos em que o desempenho foi acima do esperado. Esse conceito é desejável, mas o aluno não precisa ficar preocupado se não conseguir.

Conceito A+

O conceito A+ é atribuído aos casos em que a entrega foi perfeita. Nem problemas pontuais foram constatados.
